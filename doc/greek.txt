This input method allows to type Greek characters by simulating a
Greek keyboard layout on top of a qwerty keyboard. If your keyboard does
not havec a qwerty layout, too bad, you can still use this input method, but
the layout will not be the same as on a Greek keyboard. Here is the
correspondance table between input keys and Greek characters.
